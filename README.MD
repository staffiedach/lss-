lss is a command line tool that adds on to the deafault ls command.



INSTALLATION

1. clone this repo:
    git clone https://github.com/staffiedach/lss
    cd lss

2. compile main.cpp
    ARCH LINUX:
        run makepkg -si




USE

to run lss on the current directory, run: lss

to run lss on a specified directory, run: lss <filepath>
    if the specified path includes spaces, run </parent/'dir with spaces'>
